romewyld

Give it a resume/CV plus public metadata. Get back ranked, explained job-application leads — pulled from 11 sources, scored against you, with a transparent breakdown of why each one ranks where it does.

See a live demo dashboard Source on GitHub

How it works

resume / CV+ public metadata→ profile→ search 11 sources→ rank→ explain→ JSON / CSV / MD / HTML

Why it's different

Scored against you

It parses your skills, seniority, and history once, then ranks every posting by fit — not by keyword recency.

Transparent

Every lead shows its six-signal breakdown, matched skills, gaps, and a plain-English reason.

Wide net

Eleven sources in one pass — remote boards, Hacker Who-is-hiring, federal, and any company's careers board.

Honest AI assist

Optional Claude pass adds a fit verdict, tailored resume tweaks, and a cover-letter hook — told not to invent experience.

The score (0–100)

Profile↔posting similarity35 Skill overlap30 Title match15 Location / remote fit8 Seniority fit7 Recency5

Quick start

pip install -e ".[full]"

# resume + metadata -> ranked leads (no API keys needed)
python -m romewyld my_resume.pdf -m me.yaml

# pass BOTH a CV and a resume; add the Claude pass
python -m romewyld my_cv.pdf my_resume.docx -m me.yaml --llm

Open the generated leads.html for an interactive dashboard, or view the sample one here (built from a fictional candidate against live postings).
